# clinicaltrials
Hophacks hackathon with Adam Li, Richard Chen and Daniel Chen.

To run, pull from github.
* run virtualenv venv
* source venv/bin/activate
* pip install -r requirements.txt
* go to mysite directory and run 'python manage.py runserver'
